Al programar se hace referencia a datos que pueden ser letras o numeros

* Numeros. (50)

* Strings. ('Juan')

* Booleans. Pueder ser solo 2: true o false

Las Variables

Se manejan en todos los lenguajes de programacion

Variable es un eapacio en la memoria del ordenador (RAM) donde se almacenará un valor que podra cambiar durante la ejecucion del programa

Reglas a la hora de dar nombre a las Variables:

*Han de comenzar por una letra, simbolo $ o _

ejm: puntos
$puntos
_puntos

*Han de contener solo letras, numeros, simbolo $ y guión bajo _

*Son case sensitive.

*No deben ser palabras reservadas

Se recomienda que sea descriptivo

Iniciar la variable

Es almacenar en una variable un Valor

Se puede declarar en una linea: var puntuacion = 0;

Se puede declarar varias variables en una linea:

var puntuacion = 0; record=5000; jugador='Juan';
